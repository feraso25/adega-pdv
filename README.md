# Adega PDV

Canal de distribuição do **Adega PDV**, sistema de ponto de venda para adega de pequeno varejo
(vendas, comandas, caixa, estoque, fiados, fornecedores e relatórios automáticos no Telegram).

Este repositório serve a duas funções:

1. **Releases**: os instaladores oficiais do sistema (Windows). O aplicativo instalado se
   atualiza sozinho a partir daqui (electron-updater).
2. **`fornecedores.json`**: a lista curada de fornecedores de bebidas da região oeste da
   Grande São Paulo, com preços públicos de referência. O aplicativo baixa as atualizações
   deste arquivo sem precisar de nova versão.

O código-fonte do aplicativo não é distribuído por aqui.
